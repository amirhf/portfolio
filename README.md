# Amir Portfolio â€” Next.js (App Router) + Tailwind + MDX

A production-ready starter for your portfolio. It includes:
- Next.js App Router
- Tailwind CSS (dark-ready)
- MDX support for case studies (`app/projects/*/page.mdx`)
- Prebuilt pages: Home, Projects, Services, Writing, About, Contact
- Simple reusable components (Card, Badge, Section, Stat)

## Quickstart
```bash
pnpm i   # or npm i / yarn
pnpm dev # or npm run dev
```

## Shadcn/UI (optional)
The starter uses lightweight custom components. If you want shadcn/ui:
```bash
npx shadcn@latest init
npx shadcn@latest add button card input textarea
```

## Deploy
- Push to GitHub
- Import into Vercel
- Set your domain
